# «Шри Чайтанья-чаритамрита», Антья-лила 5.42

*эка ра̄ма̄нандера хайа эи адхика̄ра / та̄те джа̄ни апра̄кр̣та-деха та̄н̇ха̄ра* --- «Право совершать подобное, это исключительная привилегия одного Рамананды Рая, ибо по Моему разумению, его тело не материально и полностью превратилось в духовную сущность» («Шри Чайтанья-чаритамрита», Антья-лила, 5.42).
