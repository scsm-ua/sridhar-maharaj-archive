# «Шри Чайтанья-чаритамрита», Антья-лила 16.76

*чаитанйа-прабхура эи кр̣па̄ра махима̄ / брахма̄ди дева йа̄ра на̄хи па̄йа сӣма̄* — «Это торжество беспричинной милости Шри Чайтаньи Махапрабху, всю глубину которой не могут постичь даже полубоги во главе с Господом Брахмой» («Шри Чайтанья-чаритамрита», Антья-лила, 16.76).
