# «Шри Чайтанья-чаритамрита», Антья-лила 20.39

*йуга̄йитам̇ нимеш̣ен̣а, чакш̣уш̣а̄ пра̄вр̣ш̣а̄йитам / ш́ӯнйа̄йитам̇ джагат сарвам̇, говинда-вирахен̣а ме* — «Мой Господь Говиндa, в рaзлуке с Тобой кaждый миг тянется словно целaя вечность. Слезы грaдом кaтятся из моих глaз, и весь мир пуст для меня» («Шри Чайтанья-чаритамрита», Антья-лила, 20.39).

*йуга̄йитам̇ нимеш̣ен̣а, чакш̣уш̣а̄ пра̄вр̣ш̣а̄йитам / ш́ӯнйа̄йитам̇ джагат сарвам̇, говинда-вирахен̣а ме* — «Мой Господь Говинда, в разлуке с Тобой каждый миг тянется словно целая вечность.
