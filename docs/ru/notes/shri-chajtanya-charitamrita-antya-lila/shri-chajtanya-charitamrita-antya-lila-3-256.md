# «Шри Чайтанья-чаритамрита», Антья-лила 3.256

«Некогда я получила от Господа Шивы Святое Имя Господа Рамы, но сейчас благодаря общению с тобой я чувствую сильное желание повторять Святое Имя Господа Кришны» («Шри Чайтанья-чаритамрита», Антья-лила, 3.256).
