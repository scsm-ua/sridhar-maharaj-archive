# «Шри Чайтанья-чаритамрита», Антья-лила 13.131

*ниджа ш́иш̣йе кахи' говиндера мандира кара̄ила̄ / вам̇ш́ӣ, макара кун̣д̣ала̄ди 'бхӯш̣ан̣а' кари' дила* --- «Впоследствии Рагхунатха Бхатта дал указание своим ученикам построить храм для Господа Говинды и украсить Его серьгами в форме акулы, флейтой и прочими традиционными атрибутами» («Шри Чайтанья-чаритамрита», Антья-лила, 13.131).
