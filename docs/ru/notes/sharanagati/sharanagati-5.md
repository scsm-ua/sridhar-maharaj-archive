# Шаранагати 5

«Юный сын Нанды Махараджи, Шри Кришна, слышит молитвы каждого, кто принимает у Него прибежище следуя этим шести аспектам садханы» (Шрила Бхактивинод Тхакур, «Шаранагати», Введение, 5).
