# Шаранагати 3–3

*мароби ракхоби — йо иччха тохара / нитья-даса прати тува адхикара* — «Ты волен убить меня или защитить, ибо Ты — мой повелитель, а я — Твой вечный слуга» (Шрила Бхактивинод Тхакур, «Шаранагати», Атма-ниведанам (полное вручение себя Господу), 3–3).
