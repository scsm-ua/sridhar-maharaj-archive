# 袩褉械屑邪-袛褏邪屑邪-袛械胁邪-褋褌芯褌褉邪屑 16

*谢邪泻褕蹋邪-谢芯褔邪薪邪虅褕虂褉褍-胁邪褉褕蹋邪-褏邪褉褕蹋邪-泻械褕虂邪-泻邪褉褌邪薪邪屑虈\
泻芯褌蹋懈-泻邪薪蹋褌蹋褏邪-泻褉蹋褕蹋薪蹋邪-泻懈虅褉褌褌邪薪邪虅写蹋褏泄邪-写邪薪蹋写蹋邪-写褏邪虅褉邪薪邪屑\
薪泄邪褋懈-胁械褕虂邪-褋邪褉胁邪-写械褕虂邪-褏邪虅-褏褍褌邪虅褕虂邪-泻邪虅褌邪褉邪屑虈\
锌褉械屑邪-写褏邪虅屑邪-写械胁邪屑 褝胁邪 薪邪褍屑懈 谐邪褍褉邪-褋褍薪写邪褉邪屑*

芦袙 褌芯 胁褉械屑褟, 泻芯谐写邪 懈蟹 褌褘褋褟褔 谐谢邪蟹 锌芯褌芯泻邪屑懈 褌械泻谢懈 褋谢械蟹褘, 袨薪 谢懈泻芯胁邪谢, 芯褌褌芯谐芯 褔褌芯 褋褉械蟹邪谢懈 袝谐芯 锌褉械泻褉邪褋薪褘械 胁芯谢芯褋褘. 孝褘褋褟褔懈 谐芯谢芯褋芯胁 锌褉芯褋谢邪胁谢褟谢懈 袚芯褋锌芯写邪 袣褉懈褕薪褍, 懈 楔褉懈 袣褉懈褕薪邪 效邪泄褌邪薪褜褟 锌褉懈薪褟谢 锌芯褋芯褏 芯褌褉械褔械薪懈褟. 小 褌械褏 锌芯褉, 泻褍写邪 斜褘 袨薪 薪懈 褕械谢, 胁褋械, 泻褌芯 胁懈写械谢 袝谐芯 胁 芯写械卸写械 *褋邪薪薪褜褟褋懈*, 褉褘写邪谢懈 芯褌 谐芯褉褟. 携 褋泻谢芯薪褟褞褋褜 胁 锌芯褔褌械薪懈懈 锌械褉械写 锌褉械泻褉邪褋薪褘屑 袟芯谢芯褌褘屑 袚芯褋锌芯写芯屑, 袚邪褍褉邪薪谐邪 小褍薪写邪褉芯屑, 斜芯卸械褋褌胁械薪薪褘屑 胁芯锌谢芯褖械薪懈械屑 *泻褉懈褕薪邪-锌褉械屑褘*!禄 (楔褉懈谢邪 楔褉懈写褏邪褉 袦邪褏邪褉邪写卸, 芦楔褉懈 袩褉械屑邪-写褏邪屑邪-写械胁邪-褋褌芯褌褉邪屑, 16禄).
