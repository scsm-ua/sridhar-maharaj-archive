*мӯкам̇ кароти ва̄ча̄лам̇, пан̇гум̇ лан̇гхайате гирим / йат-кр̣па̄ там ахам̇ ванде, ш́рӣ-гурум̇ дӣна-та̄ран̣ам* — «Я с почтением склоняюсь перед духовным учителем, по милости которого немой может стать красноречивым оратором, а хромой — преодолеть горы» (Бхавартха Дипика).
