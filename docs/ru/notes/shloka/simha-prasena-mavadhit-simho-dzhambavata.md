*симха прасена мавадхит симхо джамбавата хатаха / шукумаарака маа родхишта тавахйеша сйамантака* — «Лев убил Прасену, лев был Джамбаваном. Не плачь, о дитя! Это жемчужина Сйамантака с вами».
