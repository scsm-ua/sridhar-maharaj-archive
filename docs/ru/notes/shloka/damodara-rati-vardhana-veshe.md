*Да̄модара-рати-вардхана-веш́е / Хари-ниш̣кут̣а-вр̣инда̄-випинеш́е* — «О царица Вриндаваны, той рощи, где наслаждается Господь Хари! О новая луна, появившаяся из океана, который олицетворяет собой царь Вришабхану!» (Шри Радхика-става Шрилы Рупы Госвами (из «Става-малы»)).
