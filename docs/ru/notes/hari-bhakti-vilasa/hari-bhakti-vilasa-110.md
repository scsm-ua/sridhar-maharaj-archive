# Хари-бхакти-виласа 110

*ман-нимиттам критам папам апи дхармайа калпате, мам анадритйа дхармо пи папам сйан мат-прабхаватах* — «Если греховные поступки совершаются ради Меня, они превращаются в благочестивые поступки, и если благочестивые дела совершаются, пренебрегая Мной, они превращаются в греховную деятельность» («Падма-пурана»; «Хари-бхакти-виласа», 110).
