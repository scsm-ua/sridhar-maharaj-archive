# Хари-бхакти-виласа 11.454

«О, лучший среди Бхарат! Святое Имя Хари может всегда присутствовать на устах только того, кто совершенным образом поклонялся Васудеве в сотнях предыдущих рождений» (Шрила Санатана Госвами, «Хари-бхакти-виласа» 11.454).
