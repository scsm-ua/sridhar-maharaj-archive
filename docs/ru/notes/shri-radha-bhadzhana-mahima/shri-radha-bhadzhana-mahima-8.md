# Шри Радха-бхаджана махима 8

«Бхактивинод, чье единственное богатство — служение Шри Радхе, смиренно молит принять его служанкой Ее лотосоподобных стоп!» (Шри Радха-бхаджана махима (Слава поклонения Шри Радхе), 8).
