# Гита-мала 1.3.3

«Я, Хари Дас, стремлюсь служить Кришне, предназначен только для Него, но как оказался в таком положении, что нигде не вижу Хари, моего Господина?» (Шрила Бхактивинод Тхакур, «Гита-мала», 1.3.3).
