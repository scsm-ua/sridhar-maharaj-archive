# Джайва-дхарма 6

«Из всего, что есть сладкого, Святое Имя --- наисладчайшее. Из всего, что есть благоприятного, Святое Имя --- самое благоприятное. Святое Имя Кришны есть вечный, полностью созревший духовный плод исполняющего желания древа ведической мудрости. О лучший из Бхригу, если кто-либо даже один раз, полный веры или с чувством безразличия, но без оскорблений, произнесет Святое Имя Господа Шри Кришны, он немедленно освободится из плена материального существования» («Хари-бхакти-виласа», Прабхаса-кханда, 11.451; «Джайва-дхарма», гл. 6).
