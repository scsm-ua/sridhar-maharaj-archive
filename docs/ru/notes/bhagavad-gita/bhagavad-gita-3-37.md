# Бхагавад-гита 3.37

«[Всевышний ответил:] Арджуна, это вожделение и гнев — порождения гуны страсти. Ненасытные, склоняющие ко всякому греху, они — величайшие враги этого мира» (Бхагавад-гита, 3.37).
