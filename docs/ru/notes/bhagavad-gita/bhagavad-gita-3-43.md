# Бхагавад-гита 3.43

«Сильнорукий Арджуна! Укрепи разум, познав то, что превыше его, — душу. Силой просветленного разума, связанного с твоим истинным «я», обуздай ум и повергни этого неукротимого врага — вожделение» (Бхагавад-гита, 3.43).
