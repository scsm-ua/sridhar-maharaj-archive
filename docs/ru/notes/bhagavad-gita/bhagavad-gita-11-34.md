# Бхагавад-гита 11.34

*дрон̣ам̇ ча бхӣшмам̇ ча джайадратхам̇ ча, карн̣ам̇ татха̄нйа̄н апи йодха-вӣра̄н / майа̄ хата̄м̇с твам̇ джахи ма̄ вйатхишт̣ха̄, йудхйасва джета̄си ран̣е сапатна̄н* — «Дрона, Бхишма, Джаядратха, Карна и другие великие воины уже уничтожены Мною. Поэтому вступи в битву без страха! Ты одержишь победу» (Бхагавад-гита, 11.34).
