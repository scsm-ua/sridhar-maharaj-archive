# 袘褏邪谐邪胁邪写-谐懈褌邪 11.34

*写褉芯薪蹋邪屑虈 褔邪 斜褏樱褕屑邪屑虈 褔邪 写卸邪泄邪写褉邪褌褏邪屑虈 褔邪, 泻邪褉薪蹋邪屑虈 褌邪褌褏邪虅薪泄邪虅薪 邪锌懈 泄芯写褏邪-胁樱褉邪虅薪 / 屑邪泄邪虅 褏邪褌邪虅屑虈褋 褌胁邪屑虈 写卸邪褏懈 屑邪虅 胁泄邪褌褏懈褕褌蹋褏邪虅, 泄褍写褏泄邪褋胁邪 写卸械褌邪虅褋懈 褉邪薪蹋械 褋邪锌邪褌薪邪虅薪* --- 芦袛褉芯薪邪, 袘褏懈褕屑邪, 袛卸邪褟写褉邪褌褏邪, 袣邪褉薪邪 懈 写褉褍谐懈械 胁械谢懈泻懈械 胁芯懈薪褘 褍卸械 褍薪懈褔褌芯卸械薪褘 袦薪芯褞. 袩芯褝褌芯屑褍 胁褋褌褍锌懈 胁 斜懈褌胁褍 斜械蟹 褋褌褉邪褏邪! 孝褘 芯写械褉卸懈褕褜 锌芯斜械写褍禄 (袘褏邪谐邪胁邪写-谐懈褌邪, 11.34).
