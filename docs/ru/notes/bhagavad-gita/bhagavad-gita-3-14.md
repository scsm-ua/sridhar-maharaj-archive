# Бхагавад-гита 3.14

*анна̄д бхаванти бхӯта̄ни, парджанйа̄д анна-самбхавах̣ / йаджн̃а̄д бхавати парджанйо, йаджн̃ах̣ карма-самудбхавах̣* — «Благодаря пище возникают тела живых существ. Для появления пищи необходим дождь, а дожди проливаются благодаря совершению жертвоприношений. Жертвоприношения же порождаются деятельностью поклоняющихся» (Бхагавад-гита, 3.14).
