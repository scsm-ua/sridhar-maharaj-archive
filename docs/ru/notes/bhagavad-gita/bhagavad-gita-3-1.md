# Бхагавад-гита 3.1

«Арджуна спросил: О Джанардана! О Кешава! Если Ты провозглашаешь, что обретение просветленного разума лучше, чем совершение деятельности, то зачем призываешь меня вступить в эту ужасную битву?» (Бхагавад-гита, 3.1).
