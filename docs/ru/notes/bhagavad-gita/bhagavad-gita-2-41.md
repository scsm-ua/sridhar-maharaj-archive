# Бхагавад-гита 2.41

«Разум тех, кто решительно посвятил себя Мне, непоколебим, ибо Я — их единственная цель. О возлюбленный потомок династии Куру, бесконечно разветвлен разум тех, кто нерешителен!» (Бхагавад-гита, 2.41).
