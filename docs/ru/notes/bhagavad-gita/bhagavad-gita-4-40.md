# 袘褏邪谐邪胁邪写-谐懈褌邪 4.40

*邪写卸薪虄邪褕虂 褔邪虅褕虂褉邪写写邪写褏邪虅薪邪褕虂 褔邪, 褋邪屑虈褕虂邪泄邪虅褌屑邪虅 胁懈薪邪褕虂泄邪褌懈 / 薪邪虅泄邪屑虈 谢芯泻芯 '褋褌懈 薪邪 锌邪褉芯, 薪邪 褋褍泻褏邪屑虈 褋邪屑虈褕虂邪泄邪虅褌屑邪薪邪褏蹋* --- 芦袧芯 薪械胁械卸写邪, 谢懈褕械薪薪褘泄 胁械褉褘 胁 袦芯懈 褋谢芯胁邪 懈 锌褉械斜褘胁邪褞褖懈泄 胁 褋芯屑薪械薪懈褟褏, 芯斜褉械褔械薪 薪邪 锌芯褉邪卸械薪懈械. 袛谢褟 褋芯屑薪械胁邪褞褖械泄褋褟 写褍褕懈 薪械褌 褋褔邪褋褌褜褟 薪懈 胁 褝褌芯泄 卸懈蟹薪懈, 薪懈 胁 褋谢械写褍褞褖械泄禄 (袘褏邪谐邪胁邪写-谐懈褌邪, 4.40).
