# 袘褏邪谐邪胁邪写-谐懈褌邪 18.2

*褕虂褉樱 斜褏邪谐邪胁邪虅薪 褍胁邪虅褔邪: 泻邪虅屑泄邪虅薪邪虅屑虈 泻邪褉屑邪薪蹋邪虅屑虈 薪泄邪虅褋邪屑虈, 褋邪薪薪泄邪虅褋邪屑虈 泻邪胁邪泄芯 胁懈写褍褏蹋 / 褋邪褉胁邪-泻邪褉屑邪-锌褏邪谢邪-褌泄邪虅谐邪屑虈, 锌褉邪虅褏褍褋 褌泄邪虅谐邪屑虈 胁懈褔邪泻褕蹋邪薪蹋邪虅褏蹋* --- 芦袩芯 屑薪械薪懈褞 屑褍写褉械褑芯胁, 芯褌泻邪蟹 芯褌 屑邪褌械褉懈邪谢懈褋褌懈褔薪芯泄 写械褟褌械谢褜薪芯褋褌懈, 芯褋薪芯胁邪薪薪芯泄 薪邪 褍写芯胁谢械褌胁芯褉械薪懈懈 胁芯卸写械谢械薪懈褟, 懈蟹胁械褋褌械薪 泻邪泻 芯褌褉械褔械薪懈械. 小胁芯斜芯写褍 卸械 芯褌 锌褉懈胁褟蟹邪薪薪芯褋褌懈 泻 锌谢芯写邪屑 谢褞斜芯泄 写械褟褌械谢褜薪芯褋褌懈 锌褉芯褋胁械褌谢械薪薪褘械 写褍褕懈 懈屑械薪褍褞褌 芯褌褉械褕械薪薪芯褋褌褜褞禄 (袘褏邪谐邪胁邪写-谐懈褌邪, 18.2).
