# Бхагавад-гита 3.20

*карман̣аива хи сам̇сиддхим, а̄стхита̄ джанака̄дайах̣ / лока-сан̇грахам эва̄пи, сампаш́йан картум архаси* — «Святой царь Джанака и другие великие правители достигли совершенства благодаря исполнению своего долга. Так же и тебе надлежит исполнять свой долг, заботясь о благе людей» (Бхагавад-гита, 3.20).
