# Бхагавад-гита 2.49

*дӯрен̣а хй аварам̇ карма, буддхи-йога̄д дханан̃джайа / буддхау ш́аран̣ам̇ анвиччха, кр̣пан̣а̄х̣ пхала-хетавах̣* — «Завоеватель богатств! С отвращением отвергни эгоистичную деятельность, обретя просветленный разум благодаря йоге. Действуй бескорыстно, ибо те, кто действует ради получения выгоды, — малодушные скупцы» (Бхагавад-гита, 2.49).
