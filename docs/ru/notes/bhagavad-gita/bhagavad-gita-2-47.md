# Бхагавад-гита 2.47

*карман̣й эва̄дхика̄рас те, ма̄ пхалеш̣у када̄чана / ма̄ карма-пхала-хетур бхӯр, ма̄ те сан̇го 'ств акарман̣и* — «Следует выполнять свой долг бескорыстно, без желания личной выгоды. Ты не должен действовать, желая насладиться плодами своего труда. Но знание о том, что плоды деятельности не принадлежат тебе, не дает тебе права пренебрегать своими обязанностями» (Бхагавад-гита, 2.47).
