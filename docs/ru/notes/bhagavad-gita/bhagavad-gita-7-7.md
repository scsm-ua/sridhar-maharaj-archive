# Бхагавад-гита 7.7

*маттах̣ паратарам̇ на̄нйат, кин̃чид асти дханан̃джайа / майи сарвам идам̇ протам̇, сӯтре ман̣и-ган̣а̄ ива* — «О завоеватель богатств, Арджуна, нет ничего выше Меня! Все сотворенное покоится на Мне, подобно драгоценным камням, нанизанным на нить» (Бхагавад-гита, 7.7).
