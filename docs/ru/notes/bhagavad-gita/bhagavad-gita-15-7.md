# Бхагавад-гита 15.7

*мамаива̄м̇ш́о джӣва-локе, джӣва-бхӯтах̣ сана̄танах̣ / манах̣-шашт̣ха̄нӣндрийа̄н̣и, пракр̣ти-стха̄ни каршати* — «Сознание (душа) — часть Меня, Моя особая сила, наделенная свободой. Будучи вечным, сознание получает во временное пользование ум и пять чувств, которые являются составными частями материи (иллюзии) — другой Моей силы» (Бхагавад-гита, 15.7).
