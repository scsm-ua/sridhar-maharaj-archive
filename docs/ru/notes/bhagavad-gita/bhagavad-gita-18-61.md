# Бхагавад-гита 18.61

«Арджуна! Всевышний пребывает в сердцах всех живых существ. По Его воле они перемещаются по кругу в механизме материального мира, сотворенном Его энергией иллюзии, *майей*». (Бхагавад-гита, 18.61).
