# Бхагавад-гита 18.61

*ӣш́варах̣ сарва-бхӯта̄на̄м̇, хр̣д-деш́е 'рджуна тиш̣т̣хати / бхра̄майан сарва-бхӯта̄ни, йантра̄рӯд̣ха̄ни ма̄йайа̄* — «Арджуна! Всевышний пребывает в сердцах всех живых существ. По Его воле они перемещаются по кругу в механизме материального мира, сотворенном Его энергией иллюзии, *майей*». (Бхагавад-гита, 18.61).
