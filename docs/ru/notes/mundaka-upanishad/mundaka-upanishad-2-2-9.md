# 袦褍薪写邪泻邪 校锌邪薪懈褕邪写 2.2.9

*褏懈褉邪薪蹋屑邪泄械 锌邪褉械 泻芯褕虂械, 胁懈褉邪写卸邪屑虈 斜褉邪褏屑邪 薪懈褕泻邪谢邪屑 / 褌邪褔 褔褏褍斜褏褉邪屑虈 写卸泄芯褌懈褕邪虅屑虈 写卸泄芯褌懈褋, 褌邪写 泄邪写 邪虅褌屑邪-胁懈写芯 胁懈写褍褏蹋 // 薪邪 褌邪褌褉邪 褋褍虅褉泄芯 斜褏邪虅褌懈 薪邪 褔邪薪写褉邪-褌邪虅褉邪泻邪屑虈, 薪械屑邪虅 胁懈写泄褍褌芯 斜褏邪虅薪褌懈 泻褍褌芯 '泄邪屑 邪谐薪懈褏蹋 / 褌邪屑 褝胁邪 斜褏邪虅薪褌邪屑 邪薪褍斜褏邪虅褌懈 褋邪褉胁邪屑虈, 褌邪褋泄邪 斜褏邪虅褋邪虅 褋邪褉胁邪屑 懈写邪屑虈 胁懈斜褏邪虅褌懈 // 斜褉邪褏屑邪懈胁械写邪屑 邪屑褉蹋褌邪屑虈 锌褍褉邪褋褌邪虅写 斜褉邪褏屑邪, 锌邪褕虂褔邪虅写 斜褉邪褏屑邪 写邪泻褕懈薪蹋邪褌邪褕虂 褔芯褌褌邪褉械薪蹋邪 / 邪写褏邪褕虂 褔芯褉写褏胁邪屑虈 褔邪 锌褉邪褋褉蹋褌邪屑虈 斜褉邪褏屑邪懈-, 胁械写邪屑虈 胁懈褕虂胁邪屑 懈写邪屑虈 胁邪褉懈褕褌蹋褏邪屑* --- 芦袛褍褏芯胁薪芯械 褑邪褉褋褌胁芯 蟹邪 锌褉械写械谢邪屑懈 屑邪褌械褉懈邪谢褜薪褘褏 芯斜芯谢芯褔械泻 锌褉械写褋褌邪胁谢褟械褌 褋芯斜芯泄 斜械褋泻褉邪泄薪械械 褋懈褟薪懈械 *袘褉邪褏屑邪薪邪*, 褋胁芯斜芯写薪芯械 芯褌 褋泻胁械褉薪褘 屑邪褌械褉懈懈. 袥褞斜芯泄 褋胁械褌 褌褉邪薪褋褑械薪写械薪褌邪谢懈褋褌褘 褋褔懈褌邪褞褌 锌褉芯褟胁谢械薪懈械屑 褝褌芯谐芯 芯褋谢械锌懈褌械谢褜薪芯 斜械谢芯谐芯 褋胁械褌邪. 袙 褌芯屑 褑邪褉褋褌胁械 薪械褌 薪褍卸写褘 胁 褋胁械褌械 褋芯谢薪褑邪 懈谢懈 谢褍薪褘, 胁 褝谢械泻褌褉懈褔械褋泻芯屑 褋胁械褌械 懈谢懈 褋胁械褌械 芯谐薪褟. 袙械褋褜 褋胁械褌 屑邪褌械褉懈邪谢褜薪芯谐芯 屑懈褉邪 --- 谢懈褕褜 芯褌斜谢械褋泻 褌芯谐芯 胁褘褋褕械谐芯 褋懈褟薪懈褟. *袘褉邪褏屑邪薪* 锌褉械斜褘胁邪械褌 胁械蟹写械: 褋锌械褉械写懈 懈 褋蟹邪写懈, 薪邪 褋械胁械褉械 懈 薪邪 褞谐械, 薪邪 蟹邪锌邪写械 懈 薪邪 胁芯褋褌芯泻械, 胁胁械褉褏褍 懈 胁薪懈蟹褍. 袠薪褘屑懈 褋谢芯胁邪屑懈, 懈蟹薪邪褔邪谢褜薪芯械 褋懈褟薪懈械 *袘褉邪褏屑邪薪邪* 锌褉芯薪懈蟹褘胁邪械褌 芯斜邪 屑懈褉邪 --- 懈 写褍褏芯胁薪褘泄, 懈 屑邪褌械褉懈邪谢褜薪褘泄禄 (芦袦褍薪写邪泻邪-褍锌邪薪懈褕邪写禄, 2.2.9--11; 锌褉懈胁芯写懈褌褋褟 胁 泻芯屑屑械薪褌邪褉懈懈 楔褉懈谢褘 袘褏邪泻褌懈胁械写邪薪褌褘 小胁邪屑懈 泻 褋褌懈褏褍 2.12 袗写懈-谢懈谢褘 芦楔褉懈 效邪泄褌邪薪褜褟-褔邪褉懈褌邪屑褉懈褌褘禄).

*褏懈褉邪薪蹋屑邪泄械 锌邪褉械 泻芯褕虂械, 胁懈褉邪写卸邪屑虈 斜褉邪褏屑邪 薪懈褕泻邪谢邪屑 / 褌邪褔 褔褏褍斜褏褉邪屑虈 写卸泄芯褌懈褕邪虅屑虈 写卸泄芯褌懈褋, 褌邪写 泄邪写 邪虅褌屑邪-胁懈写芯 胁懈写褍褏蹋 // 薪邪 褌邪褌褉邪 褋盈褉泄芯 斜褏邪虅褌懈 薪邪 褔邪薪写褉邪-褌邪虅褉邪泻邪屑虈, 薪械屑邪虅 胁懈写泄褍褌芯 斜褏邪虅薪褌懈 泻褍褌芯 '泄邪屑 邪谐薪懈褏蹋 / 褌邪屑 褝胁邪 斜褏邪虅薪褌邪屑 邪薪褍斜褏邪虅褌懈 褋邪褉胁邪屑虈, 褌邪褋泄邪 斜褏邪虅褋邪虅 褋邪褉胁邪屑 懈写邪屑虈 胁懈斜褏邪虅褌懈 // 斜褉邪褏屑邪懈胁械写邪屑 邪屑褉蹋褌邪屑虈 锌褍褉邪褋褌邪虅写 斜褉邪褏屑邪, 锌邪褕虂褔邪虅写 斜褉邪褏屑邪 写邪泻褕懈薪蹋邪褌邪褕虂 褔芯褌褌邪褉械薪蹋邪 / 邪写褏邪褕虂 褔芯褉写褏胁邪屑虈 褔邪 锌褉邪褋褉蹋褌邪屑虈 斜褉邪褏屑邪懈-, 胁械写邪屑虈 胁懈褕虂胁邪屑 懈写邪屑虈 胁邪褉懈褕褌蹋褏邪屑* --- 芦袛褍褏芯胁薪芯械 褑邪褉褋褌胁芯 蟹邪 锌褉械写械谢邪屑懈 屑邪褌械褉懈邪谢褜薪褘褏 芯斜芯谢芯褔械泻 锌褉械写褋褌邪胁谢褟械褌 褋芯斜芯泄 斜械褋泻褉邪泄薪械械 褋懈褟薪懈械 *袘褉邪褏屑邪薪邪*, 褋胁芯斜芯写薪芯械 芯褌 褋泻胁械褉薪褘 屑邪褌械褉懈懈. 袥褞斜芯泄 褋胁械褌 褌褉邪薪褋褑械薪写械薪褌邪谢懈褋褌褘 褋褔懈褌邪褞褌 锌褉芯褟胁谢械薪懈械屑 褝褌芯谐芯 芯褋谢械锌懈褌械谢褜薪芯 斜械谢芯谐芯 褋胁械褌邪. 袙 褌芯屑 褑邪褉褋褌胁械 薪械褌 薪褍卸写褘 胁 褋胁械褌械 褋芯谢薪褑邪 懈谢懈 谢褍薪褘, 胁 褝谢械泻褌褉懈褔械褋泻芯屑 褋胁械褌械 懈谢懈 褋胁械褌械 芯谐薪褟. 袙械褋褜 褋胁械褌 屑邪褌械褉懈邪谢褜薪芯谐芯 屑懈褉邪 --- 谢懈褕褜 芯褌斜谢械褋泻 褌芯谐芯 胁褘褋褕械谐芯 褋懈褟薪懈褟. *袘褉邪褏屑邪薪* 锌褉械斜褘胁邪械褌 胁械蟹写械: 褋锌械褉械写懈 懈 褋蟹邪写懈, 薪邪 褋械胁械褉械 懈 薪邪 褞谐械, 薪邪 蟹邪锌邪写械 懈 薪邪 胁芯褋褌芯泻械, 胁胁械褉褏褍 懈 胁薪懈蟹褍. 袠薪褘屑懈 褋谢芯胁邪屑懈, 懈蟹薪邪褔邪谢褜薪芯械 褋懈褟薪懈械 *袘褉邪褏屑邪薪邪* 锌褉芯薪懈蟹褘胁邪械褌 芯斜邪 屑懈褉邪 --- 懈 写褍褏芯胁薪褘泄, 懈 屑邪褌械褉懈邪谢褜薪褘泄禄 (芦袦褍薪写邪泻邪-褍锌邪薪懈褕邪写禄, 2.2.9--11; 锌褉懈胁芯写懈褌褋褟 胁 泻芯屑屑械薪褌邪褉懈懈 楔褉懈谢褘 袘褏邪泻褌懈胁械写邪薪褌褘 小胁邪屑懈 泻 褋褌懈褏褍 2.12 袗写懈-谢懈谢褘 芦楔褉懈 效邪泄褌邪薪褜褟-褔邪褉懈褌邪屑褉懈褌褘禄).
