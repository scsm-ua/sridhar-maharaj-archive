# 袗褔邪褉褜褟 胁邪薪写邪薪邪 4

*褕虂褉懈虅锌邪虅褌蹋 泻褏械褌邪褉懈 写褏邪虅屑邪, 褌蹋褏邪虅泻褍褉邪 褕虂褉懈虅 薪邪褉芯褌褌邪屑邪, 褌芯屑邪虅褌械 褌邪虅虗褏邪虅褉邪 谐褍薪蹋邪 写械泻褏懈 / 褕虂邪虅褋褌褉械褉邪 褋懈写写褏邪虅薪褌邪-褋邪虅褉邪, 褕蹋褍薪懈 谢邪虅谐械 褔邪屑邪褌泻邪虅褉邪, 泻褍褌邪虅褉泻懈泻邪 写懈褌械 薪邪虅褉械 锌褏邪虅虗泻懈* --- 芦袦褘 胁懈写懈屑, 褔褌芯 锌芯 褋胁芯懈屑 泻邪褔械褋褌胁邪屑 褌褘 薪械芯褌谢懈褔械薪 芯褌 楔褉懈谢褘 袧邪褉芯褌褌邪屑邪 袛邪褋邪 孝褏邪泻褍褉邪, 泻芯褌芯褉褘泄 薪邪 锌褉邪蟹写薪懈泻械 胁 袣褏械褌褍褉懈 袛褏邪屑械 芯褌泻褉褘谢 胁褋械屑 褋芯斜褉邪胁褕懈屑褋褟 褍写懈胁懈褌械谢褜薪褘械 蟹邪泻谢褞褔械薪懈褟 锌懈褋邪薪懈泄, 褔械屑 芯锌褉芯胁械褉谐薪褍谢 邪褉谐褍屑械薪褌褘 懈 芯斜屑邪薪 褌邪泻 薪邪蟹褘胁邪械屑褘褏 褍褔械薪褘褏禄 (袚芯锌邪谢 袚芯胁懈薪写邪 袦邪褏邪薪褌邪, 芦袗褔邪褉褜褟-胁邪薪写邪薪邪禄, 4).
