# Даинйа Прартхана Гити 1

«Когда же, наконец, Шри Чайтанья одарит меня Своей милостью? Когда же обрету я приют в тени святых стоп вайшнавов?» (Шрила Бхактивинод Тхакур, «Даинйа Прартхана Гити», 1).
