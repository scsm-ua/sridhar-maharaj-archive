# Шри Нама-Махатмья 2

*хр̣дайа хаите бале, джихва̄ра агрета чале, ш́абда-рӯпе на̄че анукш̣ан̣а / кан̣т̣хе мора бхан̇ге свара, ан̇га ка̄̐пе тхара тхара, стхира хаите на̄ па̄ре чаран̣а* — «Святое Имя исходит из моего сердца, появляется на моем языке и постоянно танцует на нем в форме трансцендентного звука. Мое дыханье прерывается, тело охватывает сильная дрожь и ноги движутся сами по себе» (Шрила Бхактивинод Тхакур, «Шаранагати», «Шри Нама-махатмья», 2).
