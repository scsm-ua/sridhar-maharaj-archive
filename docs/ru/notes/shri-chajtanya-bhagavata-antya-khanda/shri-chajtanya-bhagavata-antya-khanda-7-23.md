# «Шри Чайтанья Бхагавата», Антья-кханда 7.23

«Если Нитьянанда возьмет за руку женщину, лишенную касты или зайдет в винную лавку, Его лотосные стопы останутся незыблемым объектом поклонения, даже для Брахмы» («Шри Чайтанья-бхагавата», Антья-кханда, 7.23)
