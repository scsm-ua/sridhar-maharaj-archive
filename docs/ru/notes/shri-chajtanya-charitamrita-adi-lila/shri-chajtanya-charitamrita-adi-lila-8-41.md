# «Шри Чайтанья-чаритамрита», Ади-лила 8.41

*на̄ра̄йан̣ӣ — чаитанйера уччхиш̣т̣а-бха̄джана / та̄н̇ра гарбхе джанмила̄ ш́рӣ-да̄са-вр̣нда̄вана* — «Нараяни вечно вкушает остатки трапезы Чайтаньи Махапрабху. Шрила Вриндаван дас Тхакур появился на свет из ее лона» («Шри Чайтанья-чаритамрита», Ади-лила, 8.41).
