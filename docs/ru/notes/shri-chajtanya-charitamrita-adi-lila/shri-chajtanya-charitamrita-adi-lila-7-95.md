# 芦楔褉懈 效邪泄褌邪薪褜褟-褔邪褉懈褌邪屑褉懈褌邪禄, 袗写懈-谢懈谢邪 7.95

*褝懈 褌邪虅薪虈褉邪 胁邪虅泻泄械 邪虅屑懈 写褉蹋写蹋褏邪 胁懈褕虂胁邪虅褋邪 写褏邪褉懈', 薪懈褉邪薪褌邪褉邪 泻褉蹋褕蹋薪蹋邪-薪邪虅屑邪 褋邪薪虈泻樱褉褌邪薪邪 泻邪褉懈 / 褋械懈 泻褉蹋褕蹋薪蹋邪-薪邪虅屑邪 泻邪斜褏褍 谐邪虅芯泄邪虅泄邪, 薪邪虅褔邪虅泄邪, 谐邪虅褏懈, 薪邪虅褔懈 薪邪虅褏懈 邪虅屑懈 邪虅锌邪薪邪-懈褔褔褏邪虅泄邪* --- 芦携 褌胁械褉写芯 胁械褉褞 胁 褝褌懈 褋谢芯胁邪 袦芯械谐芯 写褍褏芯胁薪芯谐芯 褍褔懈褌械谢褟 懈 锌芯褌芯屑褍 胁褋械谐写邪 锌芯胁褌芯褉褟褞 小胁褟褌芯械 袠屑褟 袚芯褋锌芯写邪 --- 胁 芯写懈薪芯褔械褋褌胁械 懈谢懈 褋 锌褉械写邪薪薪褘屑懈. 小胁褟褌芯械 袠屑褟 袚芯褋锌芯写邪 袣褉懈褕薪褘 锌芯褉芯泄 蟹邪褋褌邪胁谢褟械褌 袦械薪褟 锌械褌褜 懈 褌邪薪褑械胁邪褌褜, 懈 锌芯褌芯屑褍 携 锌芯褞 懈 褌邪薪褑褍褞. 袧械 写褍屑邪泄, 褔褌芯 携 蟹邪褋褌邪胁谢褟褞 小械斜褟 写械谢邪褌褜 褝褌芯 --- 胁褋械 锌褉芯懈褋褏芯写懈褌 褋邪屑芯 褋芯斜芯泄禄 (芦楔褉懈 效邪泄褌邪薪褜褟-褔邪褉懈褌邪屑褉懈褌邪禄, 袗写懈-谢懈谢邪, 7.95--96).
