# «Шри Чайтанья-чаритамрита», Ади-лила 8.24

«Существуют оскорбления, о которых следует помнить при повторении мантры Харе Кришна. Повторяя „Харе Кришна" бездумно, невозможно достичь экстаза» («Шри Чайтанья-чаритамрита», Ади-лила, 8.24).
