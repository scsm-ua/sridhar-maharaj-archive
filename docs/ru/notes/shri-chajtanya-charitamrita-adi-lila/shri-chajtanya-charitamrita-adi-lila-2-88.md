# «Шри Чайтанья-чаритамрита», Ади-лила 2.88

*йа̄н̇ра бхагаватта̄ хаите анйера бхагаватта̄ / 'свайам̇-бхагава̄н'-ш́абдера та̄ха̄теи сатта̄* --- «Только Личность Бога, источник всех божественных воплощений, достоин называться *Сваям Бхагаваном*, „изначальным Господом"» («Шри Чайтанья-чаритамрита», Ади-лила, 2.88).
