# «Шри Чайтанья-чаритамрита», Ади-лила 3.91

«В сотворенном мире обитают два типа людей: обладающие божественной природой и природой демонов. Те, кто предался Господу Вишну, — люди божественного нрава, а те, кто противится Его воле, зовутся демонами» («Падма-пурана»; «Шри Чайтанья-чаритамрита», Ади-лила, 3.91).
