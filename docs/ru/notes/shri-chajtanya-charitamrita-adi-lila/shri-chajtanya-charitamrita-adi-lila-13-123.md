# «Шри Чайтанья-чаритамрита», Ади-лила 13.123

«Когда человек ест пищу, предложенную ему материалистом, его ум оскверняется, а когда ум осквернен, невозможно должным образом думать о Кришне» («Шри Чайтанья-чаритамрита», Ади-лила, 13.123).
