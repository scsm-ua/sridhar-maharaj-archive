# «Шри Чайтанья-чаритамрита», Ади-лила 6.4

*маха̄-виш̣н̣ур джагат-карта̄, ма̄йайа̄ йах̣ ср̣джатй адах̣ / тасйа̄вата̄ра эва̄йам, адваита̄ча̄рйа ӣш́варах̣* --- «Господь Адвайта Ачарья --- это воплощение Маха-Вишну, который создает материальный мир, используя силу *майи*» («Шри Чайтанья-чаритамрита», Ади-лила, 6.4).
