# «Шри Чайтанья-чаритамрита», Ади-лила 5.142

«Господь Кришна — единственный верховный повелитель, а все остальные — Его слуги. Все танцуют, подчиняясь Его воле» («Шри Чайтанья-чаритамрита», Ади-лила, 5.142).
