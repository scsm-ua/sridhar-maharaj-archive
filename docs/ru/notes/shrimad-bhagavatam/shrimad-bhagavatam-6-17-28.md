# Шримад-Бхагаватам 6.17.28

*на̄ра̄йан̣а-пара̄х̣ сарве, на куташ́чана бибхйати / сварга̄паварга-наракеш̣в, апи тулйа̄ртха-дарш́инах̣* — «Преданные, целиком посвятившие себя служению Верховной Личности Бога, Нараяне, не чувствуют страха, где бы они ни оказались. Для них нет разницы между раем, адом или освобождением, ибо таких преданных интересует только служение Господу» («Шримад-Бхагаватам», 6.17.28; «Шри Чайтанья-чаритамрита», Мадхья-лила, 9.270; 19.216).
