# Шримад-Бхагаватам 1.4.7

*катхам̇ ва̄ па̄н̣д̣авейасйа, ра̄джарш̣ер мунина̄ саха / сам̇ва̄дах̣ самабхӯт та̄та, йатраиш̣а̄ са̄тватӣ ш́рутих̣* — «Как случилось, что царь Парикшит встретил этого великого мудреца, получив возможность услышать из его уст великую песнь, заключающую в себе трансцендентную сущность Вед [„Бхагаватам"]?» («Шримад-Бхагаватам», 1.4.7).
