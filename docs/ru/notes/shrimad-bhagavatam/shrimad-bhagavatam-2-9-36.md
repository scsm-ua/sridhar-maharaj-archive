# Шримад-Бхагаватам 2.9.36

*эта̄вад эва джиджн̃а̄сйам̇, таттва-джиджн̃а̄суна̄тманах̣ / анвайа-вйатирека̄бхйа̄м̇, йат сйа̄т сарватра сарвада̄* — «Поэтому человек, который стремится обрести духовное знание, должен неустанно прямо и косвенно искать его, чтобы постичь вездесущую Абсолютную Истину» («Шримад-Бхагаватам», 2.9.36).
