# Шримад-Бхагаватам 1.2.13

*атах̣ пумбхир двиджа-ш́реш̣т̣ха̄, варн̣а̄ш́рама-вибха̄гаш́ах̣ / свануш̣т̣хитасйа дхармасйа, сам̇сиддхир хари-тош̣ан̣ам* — «Поэтому, о лучший из дваждырожденных, считается, что наивысшее совершенство, которого можно достичь, выполняя предписанные обязанности, соответствующие касте и укладу жизни, — это удовлетворить Личность Бога» («Шримад-Бхагаватам», 1.2.13).
