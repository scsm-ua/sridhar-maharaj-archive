# Шримад-Бхагаватам 11.8.21

*та̄вадж джитендрийо на сйа̄д, виджита̄нйендрийах̣ пума̄н / на джайед расанам̇ йа̄вадж, джитам̇ сарвам̇ джите расе* — «Бессмысленно обуздывать прочие чувства, покуда не подчинил себе язык. Чревоугодник не способен совладать ни с одной страстью. Напротив, обуздав чревоугодие, человек делается хозяином всех своих чувств» («Шримад-Бхагаватам», 11.8.21).
