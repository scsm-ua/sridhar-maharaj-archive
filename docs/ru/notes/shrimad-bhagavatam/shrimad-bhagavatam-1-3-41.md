# Шримад-Бхагаватам 1.3.41

*тад идам̇ гра̄хайа̄м а̄са, сутам а̄тмавата̄м̇ варам / сарва-ведетиха̄са̄на̄м̇, са̄рам̇ са̄рам̇ самуддхр̣там* — «Сняв „сливки" всех ведических писаний и летописей вселенной, Шри Вьясадева передал „Бхагаватам" своему сыну, самому высокочтимому среди осознавших себя душ» («Шримад-Бхагаватам», 1.3.41).
