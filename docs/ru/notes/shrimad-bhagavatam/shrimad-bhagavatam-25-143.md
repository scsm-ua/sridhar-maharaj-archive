# Шримад-Бхагаватам 25.143

*артхо ’йам̇ брахма-сӯтра̄н̣а̄м̇, бха̄рата̄ртха-винирн̣айах̣ / га̄йатрӣ-бха̄шйа-рӯпо ’сау, веда̄ртха-парибр̣м̇хитах̣* — «В «Шримад-Бхагаватам» разъяснен смысл «Веданта-сутры». В нем полностью раскрыты темы «Махабхараты». Здесь можно найти объяснение Брахма-гаятри, дополненное ведическим знанием во всех его тонкостях» (цитата из «Гаруда-пураны», приведенная в «Чайтанья-чаритамрите» (Мадхья-лила, 25.143)).


«В «Шримад-Бхагаватам» разъяснен смысл «Веданта-сутры». В нем полностью раскрыты темы «Махабхараты». Здесь можно найти объяснение Брахма-гаятри, дополненное ведическим знанием во всех его тонкостях» (цитата из «Гаруда-пураны», приведенная в «Чайтанья-чаритамрите» (Мадхья-лила, 25.143)).
