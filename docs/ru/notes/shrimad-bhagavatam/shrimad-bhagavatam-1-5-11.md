# Шримад-Бхагаватам 1.5.11

*тад-ва̄г-висарго джаната̄гха-виплаво, йасмин прати-ш́локам абаддхаватй апи / на̄ма̄нй анантасйа йаш́о 'н̇кита̄ни йат, ш́р̣н̣ванти га̄йанти гр̣н̣анти са̄дхавах̣* — «Но сочинение, состоящее из описаний трансцендентного величия имени, славы, форм, игр безграничного Верховного Господа, резко отличается от них. Оно составлено из трансцендентных слов, назначение которых — совершить переворот в неправедной жизни сбившейся с пути цивилизации. Такие трансцендентные произведения, даже если они несовершенны по форме, слушают, поют и принимают чистые люди, которые безукоризненно честны» («Шримад-Бхагаватам», 1.5.11).
