# «Шри Чайтанья-чаритамрита», Мадхья-лила 25.117

«Порой вместо солнца можно видеть его отражение, однако этот свет солнца не существует независимо от солнца» («Шри Чайтанья-чаритамрита», Мадхья-лила, 25.117).
