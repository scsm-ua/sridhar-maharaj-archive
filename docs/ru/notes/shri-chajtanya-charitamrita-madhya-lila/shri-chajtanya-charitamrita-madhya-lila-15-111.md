# «Шри Чайтанья-чаритамрита», Мадхья-лила 15.111

В заключение Шри Чайтанья Махапрабху сказал: «Тот, кто повторяет мантру Харе Кришна, считается вайшнавом; поэтому выражайте ему почтение» («Шри Чайтанья-чаритамрита», Мадхья-лила, 15.111).
