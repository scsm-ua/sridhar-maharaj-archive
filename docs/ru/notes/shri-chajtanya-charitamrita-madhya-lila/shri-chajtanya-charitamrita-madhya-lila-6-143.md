# «Шри Чайтанья-чаритамрита», Мадхья-лила 6.143

*брахма хаите джанме виш́ва, брахмете джӣвайа / сеи брахме пунарапи хайе йа̄йа лайа* — «Все в мироздании исходит из Абсолютной Истины, покоится на Абсолютной Истине и после разрушения снова входит в Абсолютную Истину» («Чайтанья-чаритамрита», Мадхья-лила, 6.143).
