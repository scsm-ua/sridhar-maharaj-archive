# «Шри Чайтанья-чаритамрита», Мадхья-лила 3.27

*ахо бха̄гйа, йамуна̄ре па̄илун̇ дараш́ана / эта бали' йамуна̄ра карена ставана* — «Господь сказал: „Какая удача! Наконец Я увидел Ямуну!" Приняв Гангу за Ямуну, Чайтанья Махапрабху стал возносить ей молитвы» («Шри Чайтанья-чаритамрита», Мадхья-лила, 3.27).
