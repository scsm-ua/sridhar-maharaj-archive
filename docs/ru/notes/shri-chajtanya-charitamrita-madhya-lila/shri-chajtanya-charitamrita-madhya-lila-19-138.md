# «Шри Чайтанья-чаритамрита», Мадхья-лила 19.138

«Бесчисленные живые существа скитаются по вселенной, получая тела, принадлежащие к восьми миллионам четыремстам тысячам видов жизни» («Чайтанья-чаритамрита», Мадхья-лила, 19.138).
