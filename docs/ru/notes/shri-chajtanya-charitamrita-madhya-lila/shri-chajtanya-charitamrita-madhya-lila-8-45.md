# «Шри Чайтанья-чаритамрита», Мадхья-лила 8.45

*анйера ки катха̄, а̄ми — 'ма̄йа̄ва̄дӣ саннйа̄сӣ' / а̄миха тома̄ра спарш́е кр̣ш̣н̣а-преме бха̄си* — «Хотя Я *санньяси-майявади*, то есть непреданный, просто прикоснувшись к тебе, Я тоже погрузился в океан любви к Кришне. Что же говорить о других?» («Шри Чайтанья-чаритамрита», Мадхья-лила, 8.45).
