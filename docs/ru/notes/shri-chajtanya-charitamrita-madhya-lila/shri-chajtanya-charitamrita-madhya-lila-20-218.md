# «Шри Чайтанья-чаритамрита», Мадхья-лила 20.218

«В этой вселенной Господь являет Себя в различных духовных формах. Они обитают и проводят Свои игры на семи островах, в каждой из девяти областей» («Шри Чайтанья-чаритамрита», Мадхья-лила, 20.218).
