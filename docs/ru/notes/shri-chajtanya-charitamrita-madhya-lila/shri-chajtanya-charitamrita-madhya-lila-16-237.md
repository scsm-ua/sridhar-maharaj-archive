# «Шри Чайтанья-чаритамрита», Мадхья-лила 16.237

«Наберись терпения и возвращайся домой. Не сходи с ума. Переправиться на другой берег океана материального бытия можно только постепенно» («Шри Чайтанья-чаритамрита», Мадхья-лила, 16.237).
