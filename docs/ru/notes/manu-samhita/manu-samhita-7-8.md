# Ману-самхита 7.8

*ба̄ло 'пи на̄вамантавйо мануш̣йа ити бхӯмипах̣ / махатӣ девата̄ хй эш̣а̄ нара-рӯпен̣а тиш̣т̣хати* — «Короля не следует презирать даже в детстве, потому что он уже мужчина. Он — великое Божество, явившееся в человеческой форме» («Махабхарата», 12.68.40; «Хитопадеша», 2.82; «Ману-самхита», 7.8).
