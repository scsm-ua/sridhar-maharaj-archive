# «Шри Чайтанья Бхагавата», Ади-кханда 17.152

«Тот, кто желает превзойти океан материального бытия и погрузиться в океан преданного служения, должен посвятить себя поклонению Шри Нитьянанде» («Шри Чайтанья-Бхагавата», Ади-кханда, 17.152)
