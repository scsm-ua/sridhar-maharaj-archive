# «Шри Чайтанья Бхагавата», Ади-кханда 14.85

*декхитечхи дине тина авастха̄ йа̄ха̄ра / кон ла̄дже а̄пана̄ре га̄ойайа се чха̄ра*? — «Как незначительное существо, ежедневно проходящее через три состояния, может бесстыдно побуждать людей воспевать его славу?» («Шри Чайтанья-Бхагавата, Ади-кханда, 14.85).
