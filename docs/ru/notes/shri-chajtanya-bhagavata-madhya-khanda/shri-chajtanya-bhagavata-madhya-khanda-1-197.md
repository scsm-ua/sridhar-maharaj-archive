# «Шри Чайтанья Бхагавата», Мадхья-кханда 1.197

«*Чандал* перестает быть им, если воспевает Святое Имя Кришны, а *брахман* перестает быть *брахманом*, если вступает на путь греха» («Шри Чайтанья-Бхагавата», Мадхья-кханда, 1.197).
