# Бхакти-расамрита-синдху 1.1.31

«Поразительные мистические силы, материальное наслаждение, вечное счастье осознания *Брахмана* и бесконечное блаженство служения Господу — все это приходит как результат *бхакти*, преданного служения Говинде» («Бхакти-расамрита-синдху», 1.1.31).
