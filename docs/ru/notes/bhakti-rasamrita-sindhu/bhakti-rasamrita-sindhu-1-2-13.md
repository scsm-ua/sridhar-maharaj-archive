# Бхакти-расамрита-синдху 1.2.13

«О Девариши Нарада, вся деятельность, рекомендованная священными писаниями, целью которой является удовлетворение Всевышнего, Господа Шри Хари, называется регулируемой преданностью, *вайдхи-бхакти*. Практикуя *бхакти* таким образом, человек достигает *према-бхакти*» (Шрила Рупа Госвами, «Бхакти-расамрита-синдху», 1.2.13).
